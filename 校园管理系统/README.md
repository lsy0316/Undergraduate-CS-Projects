# 智慧校园管理系统

## 📖 项目简介

智慧校园管理系统是一个基于原生 HTML/CSS/JavaScript 开发的校园信息管理平台，实现了学生、教师、课程、成绩等信息的综合管理。本项目是数据库原理课程设计作品，展示了前端开发与数据库设计的综合应用能力。

---

## 🎯 项目目标

1. 掌握 HTML/CSS/JavaScript 前端开发技术
2. 理解数据库设计原理
3. 实现完整的 CRUD 操作
4. 学习数据可视化技术

---

## 📊 数据库设计

### ER 图

```
┌─────────┐       ┌─────────┐       ┌─────────┐
│ 学生表   │       │ 选课表   │       │ 课程表   │
│ Student │───────│  SC     │───────│ Course  │
└─────────┘       └─────────┘       └─────────┘
     │                                   │
     │                                   │
     └──────────────┬────────────────────┘
                    │
              ┌─────────┐
              │ 教师表   │
              │ Teacher │
              └─────────┘
```

### 主要数据表

#### 学生表 (Student)
```sql
CREATE TABLE Student (
    Sno VARCHAR(20) PRIMARY KEY,
    Sname VARCHAR(50) NOT NULL,
    Ssex ENUM('男', '女'),
    Sage INT,
    Sdept VARCHAR(50),
    Sclass VARCHAR(30),
    Stel VARCHAR(20)
);
```

#### 课程表 (Course)
```sql
CREATE TABLE Course (
    Cno VARCHAR(20) PRIMARY KEY,
    Cname VARCHAR(100) NOT NULL,
    Ccredit INT,
    Cteacher VARCHAR(20),
    FOREIGN KEY (Cteacher) REFERENCES Teacher(Tno)
);
```

#### 选课表 (SC)
```sql
CREATE TABLE SC (
    Sno VARCHAR(20),
    Cno VARCHAR(20),
    Grade DECIMAL(5,2),
    PRIMARY KEY (Sno, Cno),
    FOREIGN KEY (Sno) REFERENCES Student(Sno),
    FOREIGN KEY (Cno) REFERENCES Course(Cno)
);
```

## 🚀 使用方法

### 1. 环境准备

- 安装 Web 服务器（如 Apache、Nginx）
- 安装 MySQL 数据库
- 导入数据库脚本

### 2. 部署步骤

```bash
# 1. 将项目文件放入 Web 服务器根目录
cp -r 校园管理系统 /var/www/html/

# 2. 导入数据库
mysql -u root -p < database.sql

# 3. 修改数据库配置
# 编辑配置文件，设置数据库连接参数

# 4. 访问系统
# 浏览器打开 http://localhost/校园管理系统/login.html
```
